# Wall Following & Visual Servoing
### Asignatura: Robótica
Autores:
- Alejandro Escorial Aparicio (alejandro.escorial@alumnos.upm.es)
- Jose Ma (jose.ma@alumnos.upm.es)

### Objetivos:
Implementar, por separado, un modelo de seguimiento de paredes y otro de seguimiento visual de una bola, ambos capaces de evitar obstáculos, sobre el robot Pioneer P3DX dado para cada caso (con o sin cámara) en el simulador [CoppeliaSim](https://www.coppeliarobotics.com/).

### Qué hay que hacer:
- Realizar una memoria en PDF para cada modelo detallando el proceso de realización, arquitectura y herramientas utilizadas.

- Mostrar resultados experimentales que demuestren el funcionamiento del sistema y añadir el código necesario como un apéndice.

- Se recomienda partir del contenido de [este](https://github.com/jdlope/robotica) repositorio, aunque se puede usar cualquier lenguaje de programación y forma de comunicación con el entorno, como la API ZMQ o ROS2.
